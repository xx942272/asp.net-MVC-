# asp.net-MVC-
使用 .NET 8 與 ASP.NET Core MVC 開發的內容管理系統，提供 Minecraft 結構資料管理、圖文內容維 護、分類篩選、關鍵字搜尋與後台管理功能。  • 使用 ASP.NET Core MVC 架構開發網站，實作 Model、View、Controller 分層設計  • 使用 Entity Framework Core 串接 SQL Server，完成資料存取與資料庫整合  • 實作後台 CRUD 功能，支援圖文資料新增、查詢、修改與刪除  • 開發關鍵字搜尋與分類篩選功能，提升資料查找效率  • 實作圖片上傳、表單處理與欄位驗證機制  • 完成資料庫設計、後端邏輯、前端頁面與 MVC 網站開發流程
